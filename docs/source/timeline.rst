.. admonition:: May 2025

    We initiated the development of the **Smart Data Center**, the foundational software infrastructure for FOOD.

    This platform aims to provide a reliable and high-performance storage architecture. It features advanced conversion tools designed to transform multi-source heterogeneous sensor data into a **unified data structure**, ensuring efficient organization and retrieval.
